## Mik az első lépések, azért hogy a git verziókezelő működjön és letöltsük az első verziót
- git init
- git config --global user.name "karolykari"
- git config --global user.email "karina.balazs92@gmail.com"
- git branch -m main
- git remote add origin https://github.com/karolykari/stilusvaltas.git
- git pull origin main

## Módosítás után szeretném feltölteni a felhőbe
- git add .
- git commit -m "Ez a második módosításom ma este"
- git push origin main
